# v0.3.1 ‚Äî Milestone 3 Phase 1: Student Session & Identity

**Date:** 2026-07-01

## Added

### Backend
- `STUDENT_COOKIE_NAME = "student_session"` in `services/auth.py`
- `create_student_token(student_id)` ‚Äî signs a 24 h JWT with `role: student`
- `require_student` FastAPI dependency ‚Äî validates `student_session` cookie, returns `student_id`
- `GET /student/me` ‚Äî returns full student identity (`student_id`, `animal_name`, `personal_code`,
  `classroom_id`, `classroom_name`) from session cookie; 401 if cookie absent or invalid
- `StudentMe` Pydantic schema

### Backend (updated)
- `POST /student/join` ‚Äî now sets `student_session` HttpOnly cookie after creating the student
- `POST /student/reconnect` ‚Äî now sets `student_session` cookie; response extended with
  `personal_code` and `classroom_name` (previously only returned `animal_name` + `classroom_id`)
- `StudentReconnectResponse` schema extended with `personal_code` and `classroom_name`
