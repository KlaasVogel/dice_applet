# v0.1.0 — Milestone 1: Database Schema & Backend Skeleton

**Date:** 2026-06-16

## Added

- FastAPI backend skeleton in `src/dice_applet/` (`config`, `db`, `routers`, `schemas`, `services`)
- SQLAlchemy async models: `Classroom`, `Student`, `StudentDataset`, `Measurement`
- Alembic migration setup (async-aware `env.py`) with initial schema revision
- Endpoints:
  - `GET /health`
  - `POST /teacher/login`, `DELETE /teacher/logout`, `POST /teacher/classrooms`, `GET /teacher/classrooms`
  - `POST /student/join`, `POST /student/reconnect`
  - `GET /classroom/{id}` (teacher-auth required)
- Teacher session auth via signed JWT cookie; student identity via personal code lookup
- Animal-name + personal-code generation services
- `Dockerfile`, `.env.example`, `pyproject.toml`
- `tests/test_health.py` (pytest-asyncio + httpx)
